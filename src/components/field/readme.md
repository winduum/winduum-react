# [Field](https://winduum.dev/docs/components/field.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Field } from '@/components/ui/field'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Field } from 'winduum-react/src/components/field'
```

### Docs
Visit [docs](https://winduum.dev/docs/components/field.html) to learn more and see usage examples.
