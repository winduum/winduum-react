# [Progress](https://winduum.dev/docs/ui/progress.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Progress } from '@/components/ui/progress'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Progress } from 'winduum-react/src/components/progress'
```

### Docs
Visit [docs](https://winduum.dev/docs/ui/progress.html) to learn more and see usage examples.
