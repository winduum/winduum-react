# [Heading](https://winduum.dev/docs/ui/heading.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Heading } from '@/components/ui/heading'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Heading } from 'winduum-react/src/components/heading'
```

### Docs
Visit [docs](https://winduum.dev/docs/ui/heading.html) to learn more and see usage examples.
