# [Notice](https://winduum.dev/docs/ui/notice.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Notice } from '@/components/ui/notice'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Notice } from 'winduum-react/src/components/notice'
```

### Docs
Visit [docs](https://winduum.dev/docs/ui/notice.html) to learn more and see usage examples.
