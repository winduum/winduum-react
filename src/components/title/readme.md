# [Title](https://winduum.dev/docs/ui/title.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Title } from '@/components/ui/title'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Title } from 'winduum-react/src/components/title'
```

### Docs
Visit [docs](https://winduum.dev/docs/ui/title.html) to learn more and see usage examples.
