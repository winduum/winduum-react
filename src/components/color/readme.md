# [Control](https://winduum.dev/docs/ui/color.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Color } from '@/components/ui/color'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Color } from 'winduum-react/src/components/color'
```

### Docs
Visit [docs](https://winduum.dev/docs/ui/color.html) to learn more and see usage examples.
