# [Badge](https://winduum.dev/docs/ui/badge.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Badge } from '@/components/ui/badge'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Badge } from 'winduum-react/src/components/badge'
```

### Docs
Visit [docs](https://winduum.dev/docs/ui/badge.html) to learn more and see usage examples.
