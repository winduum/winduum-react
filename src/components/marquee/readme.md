# [Marquee](https://winduum.dev/docs/components/marquee.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Marquee } from '@/components/ui/marquee'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Marquee } from 'winduum-react/src/components/marquee'
```

### Docs
Visit [docs](https://winduum.dev/docs/components/marquee.html) to learn more and see usage examples.
