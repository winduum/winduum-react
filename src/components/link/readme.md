# [Link](https://winduum.dev/docs/ui/link.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Link } from '@/components/ui/link'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Link } from 'winduum-react/src/components/link'
```

### Docs
Visit [docs](https://winduum.dev/docs/ui/link.html) to learn more and see usage examples.
