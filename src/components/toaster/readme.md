# [Toaster](https://winduum.dev/docs/components/toaster.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Toaster } from '@/components/ui/toaster'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Toaster } from 'winduum-react/src/components/toaster'
```

### Docs
Visit [docs](https://winduum.dev/docs/components/toaster.html) to learn more and see usage examples.
