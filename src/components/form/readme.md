# [Form](https://winduum.dev/docs/components/form.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Form } from '@/components/ui/form'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Form } from 'winduum-react/src/components/form'
```

### Docs
Visit [docs](https://winduum.dev/docs/components/form.html) to learn more and see usage examples.
