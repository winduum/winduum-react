# [Drawer](https://winduum.dev/docs/components/drawer.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Drawer, DrawerContent, DrawerScroller } from '@/components/ui/drawer'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Drawer, DrawerContent, DrawerScroller } from 'winduum-react/src/components/drawer'
```

### Docs
Visit [docs](https://winduum.dev/docs/components/drawer.html) to learn more and see usage examples.
