# [Popover](https://winduum.dev/docs/components/popover.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Popover, PopoverContent } from '@/components/ui/popover'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Popover, PopoverContent } from 'winduum-react/src/components/popover'
```

### Docs
Visit [docs](https://winduum.dev/docs/components/popover.html) to learn more and see usage examples.
