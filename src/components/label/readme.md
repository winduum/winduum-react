# [Label](https://winduum.dev/docs/ui/label.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Label } from '@/components/ui/label'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Label } from 'winduum-react/src/components/label'
```

### Docs
Visit [docs](https://winduum.dev/docs/ui/label.html) to learn more and see usage examples.
