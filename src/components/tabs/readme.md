# [Tabs](https://winduum.dev/docs/components/tabs.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Tabs, TabsList } from '@/components/ui/tabs'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Tabs, TabsList } from 'winduum-react/src/components/tabs'
```

### Docs
Visit [docs](https://winduum.dev/docs/components/tabs.html) to learn more and see usage examples.
