# [Dialog](https://winduum.dev/docs/components/dialog.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Dialog, DialogContent } from '@/components/ui/dialog'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Dialog, DialogContent } from 'winduum-react/src/components/dialog'
```

### Docs
Visit [docs](https://winduum.dev/docs/components/dialog.html) to learn more and see usage examples.
