# [Carousel](https://winduum.dev/docs/components/carousel.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Carousel } from '@/components/ui/carousel'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Carousel } from 'winduum-react/src/components/carousel'
```

### Docs
Visit [docs](https://winduum.dev/docs/components/carousel.html) to learn more and see usage examples.
