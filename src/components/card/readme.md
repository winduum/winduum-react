# [Card](https://winduum.dev/docs/components/card.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Card } from '@/components/ui/card'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Card } from 'winduum-react/src/components/card'
```

### Docs
Visit [docs](https://winduum.dev/docs/components/card.html) to learn more and see usage examples.
