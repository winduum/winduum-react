# [Breadcrumb](https://winduum.dev/docs/components/breadcrumb.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Breadcrumb } from '@/components/ui/breadcrumb'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Breadcrumb } from 'winduum-react/src/components/breadcrumb'
```

### Docs
Visit [docs](https://winduum.dev/docs/components/breadcrumb.html) to learn more and see usage examples.
