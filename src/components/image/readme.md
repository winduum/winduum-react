# [Image](https://winduum.dev/docs/components/image.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Image } from '@/components/ui/image'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Image } from 'winduum-react/src/components/image'
```

### Docs
Visit [docs](https://winduum.dev/docs/components/image.html) to learn more and see usage examples.
