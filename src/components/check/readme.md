# [Check](https://winduum.dev/docs/ui/check.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Check } from '@/components/ui/check'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Check } from 'winduum-react/src/components/check'
```

### Docs
Visit [docs](https://winduum.dev/docs/ui/check.html) to learn more and see usage examples.
