# [Compare](https://winduum.dev/docs/components/compare.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Compare } from '@/components/ui/compare'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Compare } from 'winduum-react/src/components/compare'
```

### Docs
Visit [docs](https://winduum.dev/docs/components/compare.html) to learn more and see usage examples.
