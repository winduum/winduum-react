# [Info](https://winduum.dev/docs/ui/info.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Info } from '@/components/ui/info'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Info } from 'winduum-react/src/components/info'
```

### Docs
Visit [docs](https://winduum.dev/docs/ui/info.html) to learn more and see usage examples.
