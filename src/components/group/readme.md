# [Group](https://winduum.dev/docs/components/group.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Group } from '@/components/ui/group'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Group } from 'winduum-react/src/components/group'
```

### Docs
Visit [docs](https://winduum.dev/docs/components/group.html) to learn more and see usage examples.
