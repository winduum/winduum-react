# [Button](https://winduum.dev/docs/ui/control.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Control } from '@/components/ui/control'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Control } from 'winduum-react/src/components/control'
```

### Docs
Visit [docs](https://winduum.dev/docs/ui/control.html) to learn more and see usage examples.
