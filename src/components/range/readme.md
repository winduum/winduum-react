# [Range](https://winduum.dev/docs/ui/range.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Range } from '@/components/ui/range'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Range } from 'winduum-react/src/components/range'
```

### Docs
Visit [docs](https://winduum.dev/docs/ui/range.html) to learn more and see usage examples.
