# [Tooltip](https://winduum.dev/docs/components/tooltip.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Tooltip } from '@/components/ui/tooltip'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Tooltip } from 'winduum-react/src/components/tooltip'
```

### Docs
Visit [docs](https://winduum.dev/docs/components/tooltip.html) to learn more and see usage examples.
