# [Toast](https://winduum.dev/docs/components/toast.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Toast, ToastContent } from '@/components/ui/toast'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Toast, ToastContent } from 'winduum-react/src/components/toast'
```

### Docs
Visit [docs](https://winduum.dev/docs/components/toast.html) to learn more and see usage examples.
