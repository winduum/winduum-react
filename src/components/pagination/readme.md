# [Pagination](https://winduum.dev/docs/components/pagination.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Pagination } from '@/components/ui/pagination'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Pagination } from 'winduum-react/src/components/pagination'
```

### Docs
Visit [docs](https://winduum.dev/docs/components/pagination.html) to learn more and see usage examples.
