# [Button](https://winduum.dev/docs/ui/button.html)

## Installation
By default, we recommend copying and pasting the code from this directory to your project and remapping the imports to local.

```js
import { Button } from '@/components/ui/button'
```
### NPM
Alternatively, you can also import directly from `npm` so you can leverage updates.
```shell
npm i winduum-react
```

```js
import { Button } from 'winduum-react/src/components/button'
```

### Docs
Visit [docs](https://winduum.dev/docs/ui/button.html) to learn more and see usage examples.
